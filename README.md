# Sam Rhea's Blog

This repository contains the content and configuration for [my personal blog](https://blog.samrhea.com).

## 🏗️ Cloudflare Pages

This blog runs on [Cloudflare Pages](https://pages.cloudflare.com/).

## 💡 Gatsby Lumen Theme

The site uses the [Lumen Gatsby starter theme](https://www.gatsbyjs.com/starters/alxshelepenok/gatsby-starter-lumen), maintained by [this team](https://github.com/alxshelepenok/gatsby-starter-lumen#contributors).

### Edits to Lumen
These are the edits I have made from the theme default above beyond the standard config file changes.

#### Remove Scroll Into View

[fb39748](https://github.com/TownLake/blog-samrhea/commit/bce047af82ef2275c6a11a2eeccb09f2d8275466) removes the effect where the posts would scroll into view on mobile and skip over the main links.

#### Hide Book Posts
[3020e63](https://github.com/TownLake/blog-samrhea/commit/0d6c15d12576e91b1ae0438d269b76ecf5d1074c) hides `Reading` posts from the main blog feed.

#### GitHub Table Format
[fd27a90](https://github.com/TownLake/blog-samrhea/commit/49c932dc979d53564fd3f2371fce1c76dfaa61a2) adds styling to the default table to mirror that of GitHub, which I prefer.

#### Remove Read More Link
[fd27a90](https://github.com/TownLake/blog-samrhea/commit/fd27a904a988e3f4f009f0ed3f94b09a551ef7e5) removes the `Read More` text which I found redundant.
